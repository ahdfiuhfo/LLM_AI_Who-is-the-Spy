安装
在使用本项目前，你需要安装 openai 库。你可以通过以下命令安装：

bash
pip install openai

如果这个命令有误，请上网搜索正确的安装方法。

使用
克隆此项目到你的本地机器：
bash
git clone <repository-url>

进入项目目录：
bash
cd <project-directory>

安装依赖项：
bash
pip install -r requirements.txt

运行程序：
bash
python main.py

程序的可读性比较差，希望你不要介意！
